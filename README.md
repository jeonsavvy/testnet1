# JEON CHAN HYUK

Personal portfolio playground built with **Next.js 14** & **Three.js** (R3F).
Just testing some design concepts and 3D interactions.

[**Live Site**](https://jeonchanhyuk.pages.dev/)

## Stack
- **Framework:** Next.js 14 (App Router)
- **3D:** React Three Fiber, Drei
- **Styling:** Vanilla CSS (Modules)

## Run
```bash
npm install
npm run dev
```
